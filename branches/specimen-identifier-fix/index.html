This is a solution FHIR Implementation Guide (IG) for the Rwandan VLSM data exchange.

This data elements in ths FHIR IG have been informed by the Rwandan VLSM minimum dataset (MDS).

  • This FHIR IG is not intended for production use until all partner reviews have been completed.
  • The API contract for VLSM Viral Load (VL) v1.1 was used as the basis for the MDS data elements.

The FHIR Resource References Guide illustrates and describes the various resources and their references in this implementation.

  • The FHIR Resource References Guide will be embedded into this FHIR IG at a later stage.

Testing

The Rwanda VLSMS Postman Collection provides a few sample bundles that can be used during developer testing. It also allows a developer to create multiple test cases using the pre-request script coming with each Postman request.



Download

You can also download:

  • this entire guide,
  • the definition resources in json, xml, ttl, or csv format, or
  • the example resources in json, xml or ttl format.

The source code for this Implementation Guide can be found on OpenHIE GitHub.
